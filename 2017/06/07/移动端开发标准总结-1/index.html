移动端开发标准总结---1

2017.06.07 bobo-33  热度 ℃

###Welcome to use MarkDown

  • height固定

    1. fixed-top:48px
    2. fixed-bottom:49px
    3. 列表型条性格用–line-height：50px
    4. 列表型条性格间距用–min-height:12px

  • border

    background-image: linear-gradient(180deg,#ddd,#ddd 50%,transparent 50%),linear-gradient(0deg,#ddd,#ddd 50%,transparent 50%);
    background-size: 100% 1px,100% 1px;
    background-repeat: no-repeat;
    background-position: top,bottom;



代替

    border：1px solid #ccc;（区分几倍屏幕，当大于一倍是边框就会很粗）

  • 字体

    1. 字体一般为16px
    2. 字体图标用阿里字体库（显示的都是能用上的）

    3. 文字可用—–

      • 字体

        :before{content:"￥"}

      • 属性

        a.content:after {
   content: attr(href);
}

<a class="content" href="http://www.zhangxinxu.com/">此标签的href值是：</a>

      • 图片

        p a[href $=".pdf"]:after {
    content:url(../image/icon_pdf.png);
}
p a[rel = "external"]:after {
    content:url(../image/icon_link.png);
}

  • 宽度

    width: calc(50% - 40px);——(width-(padding+border)*2-margin)——需要有空格

sass---总结

2017.06.17 bobo-33  热度 ℃

  • css预处理器简介

    css预处理器为css增加编程特性，无需考虑浏览器兼容性。

css预处理器语言更加简洁，可读性强，适应性强，易于维护。

  • sass特点（只能使用sass老语法规则）

    用更简洁的语法清晰，结构化描述样式。

用ruby语言

缩进式语法

不带{}和；

  • scss语法与普通css一致（使用了sass新语法规则）

  • 安装

    安装ruby （ruby -v---安装成功）

安装sass  （gem install sass，升级sass版本gem update sass，查看sass版本sass -v）

  • 引用sass

    先写sass文件

命令行编译：sass --watch <要编译的Sass文件路径>/style.scss:<要输出CSS文件路径>/style.css

用watch本地文件一旦发生变化，css就会改变

  • sass不同样式风格输出

    1. 嵌套输出方式 nested

2. 展开输出方式 expanded  

3. 紧凑输出方式 compact 

4. 压缩输出方式 compressed

  • sass声明变量

    组成：（1）声明变量符$;（2）变量名称；（3）变量值

如：

    $color: red;

  • 语法

    1. ！default—默认声明

    2. 嵌套

      选择器嵌套：

      nav {
  a {
    color: red;

    header & {
      color:green;
    }
  }  
}

      属性嵌套

      .box {
  border: {
   top: 1px solid red;
   bottom: 1px solid green;
  }
}

      伪类嵌套

      .box{
  &:before {
    content:"伪元素嵌套";
  }
}

  • 混合宏

    声明：

        @mixin border-radius{
            -webkit-border-radius: 3px;
            border-radius: 3px;
        }

调用：

        button {
            @include border-radius;
        }

    1. 混合宏变量（不带值）

      @mixin border-radius($radius){
  -webkit-border-radius: $radius;
  border-radius: $radius;
}

.box {
  @include border-radius(3px);
}

    2. 混合宏变量（带值）

      @mixin border-radius($radius:3px){
  -webkit-border-radius: $radius;
  border-radius: $radius;
}

.box {
  @include border-radius;
}

    3. 混合宏多个参数（不带值）

      @mixin size($width,$height){
  width: $width;
  height: $height;
}

.box-center {
  @include size(500px,300px);
}

    4. 混合宏的缺点

      容易产生冗余代码

    5. 继承

      .btn {
  border: 1px solid #ccc;
  padding: 6px 10px;
  font-size: 14px;
}

.btn-primary {
  background-color: #f36;
  color: #fff;
  @extend .btn;
}

    6. 占位符—%（当不调用时不显示不产生冗余代码）

      %mt5 {
  margin-top: 5px;
}

.btn {
  @extend %mt5;
}

    7. 数据类型

      • 数字: 如，1、 2、 13、 10px；

      • 字符串：有引号字符串或无引号字符串，如，”foo”、 ‘bar’、 baz；

      • 颜色：如，blue、 #04a3f9、 rgba(255,0,0,0.5)；

      • 布尔型：如，true、 false；

      • 空值：如，null；

      • 值列表：用空格或者逗号分开，如，1.5em 1em 0 2em 、 Helvetica, Arial, sans-serif。

Sass运算

加法运算

  • 变量或属性中都可以运算

  • 不同类型单位报错（如ex,em,rem—-相对单位）

    .width{            
    20px+2em;
}

减法运算

  • 法则同上

乘法运算

  • 支持多种单位

  • 一个单位有多种值报错

    .width{        
    20px*10px;    
    }

  • 同一个单位写一个值

    .width{
    20px*10;
}

  • 不同类型单位报错

    .width{
    20px*2em;
}

除法运算

  • 法则同乘法

  • 单独使用“/”将不会生效，应该加上“（）”

    .width{
    (960px / 10);
}

  • 当除法是数学表达式中的一部分，不加“（）”也会生效

    .width{
    960px / 10+10;
}

变量计算

计算列宽：

$col-width: 60px;
$col-gap: 20px;

@for $i from 1 through 12 {
    .col-#{$i}{
        width:$col-width*$i+$col-gap*($i-1);
    }
}

数字计算

  • 数字运算包括前面介绍的：加法、减法、乘法和除法等运算。而且还可以通过括号来修改他们的运算先后顺序。

    .box {
  width: ((220px + 720px) - 11 * 20 ) / 12 ;  
}

颜色计算

p {
  color: #010203 + #040506;
}

字符运算

通过“+”链接

$content: "Hello" + "" + "Sass!";
    .box:before {
      content: " #{$content} ";
    }

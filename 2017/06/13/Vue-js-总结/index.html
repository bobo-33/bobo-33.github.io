Vue.js—–总结

vue.js特点

  • 结合了react和angular的优点—–react：组件化，angular：灵活指令和页面操作方法

  • 基于MVVM模式的数据双向绑定

  • IE8及IE8以下不兼容

vue.js–组件

  • html

    <template></template>

  • css

    <style></style>

  • javascript

    <script></script>

vue重要选项

  • data

  • methods

  • watch(监听)

vue对象里通过指令进行html关联

###数据渲染

  • v-text

  • v-html

###控制模块隐藏

  • v-if

  • v-show

区别：v-if是直接渲染DOM元素，v-show是通过css控制

###循环列表

  • v-for

###事件绑定

  • v-on:事件

如：

<button v-on:click="dothis"></button>

可简化为：

<button @click="dothis"></button>

script:

new Vue({
    methods:{
        dothis:function( something ){

        }
    }
})

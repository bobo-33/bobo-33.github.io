:before和:after用法总结

2017.06.15 bobo-33 programming  热度 ℃

伪类和伪元素

  • 伪类：向某选择器添加特殊效果

  • 伪元素：将特殊效果添加到某选择器

    span:hover//伪类
span::before//伪元素

为了兼容性，将伪类和伪元素都写成单冒号

常见的伪类和伪元素

  • 伪类

    1. :link

    2. :visited

    3. :hover

    4. :active

    5. :target

    6. :not(s)

    7. :focus

  • 伪元素

    1. :before

    2. :after

    3. :first-letter

    4. :first-line

    5. :selection

伪元素的兼容性

  • Chrome 2+,
  • Firefox 3.5+ (3.0 had partial support),
  • Safari 1.3+,
  • Opera 9.2+,
  • IE8+ (with some minor bugs),
  • 几乎所有的移动浏览器。

伪元素:before和:after定义

  • ：after—-在备选元素后面加入内容

  • ：before—-在备选元素前面加入内容

伪元素:before和:after——-content几个属性用法

  • 添加字符串

    a:before{
    content:"文字"
}

  • 添加href属性

    a:before{
    content: "("attr(href)")"
}

  • 引用媒体文件

    a:before{
    content:url(bg.jpg);
}

  • 自定义

    a:before{
    content:attr(data-title);
}

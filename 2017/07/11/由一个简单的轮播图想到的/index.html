博主研究了好半天，觉得有一个一直不明白的问题终于懂啦~那就是shift(),unshift(),pop(),push()

辨析数组方法——shift(),unshift(),pop(),push()

  • shift()—–shift弹出数组第一个元素，返回被弹出的元素值

  • unshift()——unshift 在数组第一个元素前插入元素

  • pop()—–pop函数删除数组末尾元素

  • push()—–push在数组末尾添加元素

在上个栗子中，每隔1000ms轮播图img的src值是arr.pop()的返回值，如此循环。

shift()方法

小实例：

// 使用shift弹出数组第一个元素
// 返回被弹出的元素值
var tmp = ['a','b','c'];
var val = tmp.shift();
alert(val); // a
alert(tmp); // b,c

如果是一个空数组：

// 使用shift弹出数组第一个元素
// 返回被弹出的元素值
var tmp = [];
var val = tmp.shift();
alert(val); // undefined
alert(tmp); // 空

unshift()方法

unshift // 在数组第一个元素前插入元素    
// 使用unshift在数组第一个元素前插入元素
// 返回数组长度
var tmp = ['a','b'];
var len = tmp.unshift('c');
alert(len); // 3
alert(tmp); // c,a,b

也可以一次插入多个元素，顺序依次从左边排起

// 使用unshift在数组第一个元素前插入元素
// 返回数组长度
var tmp = ['a','b'];
var len = tmp.unshift('c','d');
alert(len); // 4
alert(tmp); // c,d,a,b

push()方法

跟unshift相反，push在数组末尾添加元素，返回添加元素以后的数组长度

// 使用push在数组末尾添加多个元素
// 返回数组最新长度
var tmp = ['a','b','c'];
var len = tmp.push('d');
alert(len); // 4
alert(tmp); // a,b,c,d

也可以一次添加多个元素

// 使用push在数组末尾添加多个元素
// 返回数组最新长度
var tmp = ['a','b','c'];
var len = tmp.push('d','e','f');
alert(len); // 6
alert(tmp); // a,b,c,d,e,f

pop()方法

跟shift相反，pop弹出的是数组末尾元素，返回被弹出的元素值

// 使用pop弹出数组末尾元素
// 返回被弹出的元素值
var tmp = ['a','b','c'];
var val = tmp.pop();
alert(val); // c
alert(tmp); // a,b

如果数组为空，返回undefined

// 使用pop弹出数组末尾元素
// 返回被弹出的元素值
var tmp = [];
var val = tmp.pop();
alert(val); // undefined
alert(tmp); // 空
